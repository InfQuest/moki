## About

Turn any YouTube video into your personal language classroom. AI-powered subtitle translation, phrase extraction, and smart vocabulary review.

## Features

- **AI Translation** - Real-time translation of YouTube subtitles, supporting 56 language pairs with precise understanding
- **Smart Phrase Extraction** - Automatically identify key phrases and expressions in videos to master authentic usage
- **Vocabulary Collection** - Save words and example sentences with one click to build your personal vocabulary library
- **Anki Export** - Seamlessly export your vocabulary to Anki for efficient spaced repetition review

## Supported Languages

**Target Languages (7)**: English, Chinese, Japanese, Korean, Spanish, French, German

**Native Languages (8)**: English, Simplified Chinese, Traditional Chinese, Japanese, Korean, Spanish, French, German

## Installation

### From Chrome Web Store (Recommended)

Visit the [Chrome Web Store](https://chromewebstore.google.com/detail/moki-learning-language-in/anocpbjiamfpdnenlgogpcjjfcnccklk) and click "Add to Chrome".

### Manual Installation (Developer Mode)

1. Download the latest release from [Releases](https://github.com/InfQuest/moki/releases)
2. Unzip the downloaded file
3. Open Chrome and go to `chrome://extensions/`
4. Enable "Developer mode" in the top right
5. Click "Load unpacked" and select the unzipped folder

## How It Works

1. **Install Extension** - One-click install Moki extension from Chrome Web Store
2. **Open Video** - Choose any YouTube video you like to start learning
3. **Enable Learning Mode** - Click the logo under the video to enable learning mode
4. **Collect Knowledge** - Save words and phrases with full video context
